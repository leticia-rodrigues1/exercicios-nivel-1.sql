Exercícios SQL - Nível 1

Este repositório reúne exercícios práticos desenvolvidos em SQL Server com foco nos fundamentos da análise de dados.

🎯 Objetivo

Consolidar conhecimentos básicos em SQL por meio de prática estruturada, aplicando consultas em cenários simulados de negócio.

🧠 Conteúdos abordados:

SELECT

WHERE

JOIN (INNER JOIN)

GROUP BY

ORDER BY

Funções agregadas (COUNT, SUM)


📁 Estrutura do projeto

Os exercícios estão organizados em pastas por nível:

Exercício 01 → Total gasto por cliente (GROUP BY + SUM)

Exercício 02 → Quantidade de pedidos por cliente (COUNT)

Exercício 03 → Filtro de pedidos acima de determinado valor (WHERE)

Exercício 04 → Ordenação de dados (ORDER BY)

Exercício 05 → Classificação de dados (CASE WHEN)
